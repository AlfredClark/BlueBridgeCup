# BEGIN-2 序列求和

## 资源限制

>时间限制：1.0s  内存限制：256.0MB

## 问题描述

> 求1+2+3+...+n的值。

## 输入格式

> 输入包括一个整数n。

## 输出格式

> 输出一行，包括一个整数，表示1+2+3+...+n的值。

## 样例输入

> 4

## 样例输出

> 10

## 数据规模与约定

> 1 <= n <= 1,000,000,000。 

## 解题思路

> 使用基础数学知识

## 通过代码

```cpp
#include <iostream>

using namespace std;

int main() {
    long long N = 0;
    cin >> N;
    cout << (N + 1) * N / 2;
    return 0;
}
```
